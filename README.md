# DAA Detective: Crack the Case with Algorithms

An interactive, game-style **Design and Analysis of Algorithms (DAA)** project built using only **HTML, CSS, and JavaScript**.

Players solve algorithm-driven detective cases with animated visualizations, scoring, and progress tracking.

## Live Concept

`DAA Detective` turns algorithm learning into case-solving missions:
- Pick a case from the home dashboard
- Run the algorithm visualization step-by-step
- Understand logic, complexity, and conclusion
- Solve the case to unlock points and rank up

## Key Features

- Cinematic detective-themed UI with particle background and animated overlays
- 15 interactive algorithm case files
- Real-time logs during algorithm execution
- Time and space complexity shown per case
- Difficulty indicator and points per mission
- Score + solved cases + rank system
- Progress persistence using browser `localStorage`
- Adjustable global animation speed control
- Fully client-side project (no backend, no framework)

## Algorithms Covered

1. Dijkstra's Shortest Path
2. 0/1 Knapsack (Dynamic Programming)
3. Merge Sort (Divide & Conquer)
4. Backtracking (N-Queens Variant)
5. BFS (Graph Traversal)
6. Kruskal's MST
7. Branch and Bound (TSP)
8. Huffman Encoding
9. Rat in a Maze (Backtracking)
10. Quick Sort
11. Floyd-Warshall (All-Pairs Shortest Path)
12. Longest Common Subsequence (LCS)
13. Topological Sort
14. Binary Search
15. Rabin-Karp String Matching

## Tech Stack

- HTML5
- CSS3 (custom styling, gradients, animations)
- Vanilla JavaScript (DOM manipulation, visualization logic)
- Browser Local Storage (progress and score)

## How to Run Locally

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` directly in your browser.

No installation, build step, or dependencies required.

## Learning Value

This project helps students:
- understand core DAA paradigms (Greedy, DP, Graph, Backtracking, Divide & Conquer)
- connect theory with visual execution
- compare complexities in practical scenarios
- learn through interactive storytelling instead of static notes
